Was ist das?

JS= JavaScript, eine Skriptsprache für dynamische Interpretation im Webbrowser

JScript= Erweiterung für JavaScript (Microsoft Features)

VB-Script= Visual Basic (mit VBA = Visual Basic for Applications)

Java= Ist die Kompilierte Programmiersprache. Sie hat keine Bezüge zu JavaScript.
