Was ist CSS (kurz erklärt)?
Erklärung Cascading Style Sheets (CSS) ist eine Programmiersprache, die ermöglicht, das Design von HTML Websites zu bestimmen.
Zum Design gehört die Farbe, Ausrichtungen, Bilder etc.
CSS entstand Mitte der 90er-Jahre und gilt heute als die Standard-Stylesheet-Sprache im World Wide Web.

Was ist Boxmodell?
Erklärung Das CSS-Box-Modell ist eine Reihe von Regeln, die definieren, wie jedes Element auf einer Webseite gerendert wird. Nach dem Box-Modell ist jedes Element einer Seite eine rechteckige Box mit einem Inhalt, einer Breite und einer Höhe. Ein Inhalt muss aber nicht zwingend vorhanden sein. Ohne Angaben ist diejenige Box auch nicht da.

Fragen zum Boxmodell
Wie sind die Pixel, wenn die Content Box 100px hat, die padding-top 3px height und die padding-bottom 2px height? Content height sind 100, padding-top 3 und padding-bottom 2, das ist 100+3+2=105px
Welche Boxen gibt es? Content Box, Padding Box, Border Box, Margin Box
