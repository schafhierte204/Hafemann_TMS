Hintergründe in CSS



Hintergründe einfarbig Hintergründe können mit
> background: darkgrey; <
im CSS erstellt werden

Einfarbig und Transparent Hintergrundbilder können mit
> background: green; opacity: 0.3; <
im CSS erstellt werden

Bild Mit
> background-image: url("beispiel.jpg");
kann man ein Bild/Foto als Hindergrund einstellen

die sich nicht wiederholen Hintergrundbilder werden meist einfach horizontal und vertikal wiederholt um den Hintergrund zu füllen. Dies kann man mit
> background-image: url("bg.jpg"); background-repeat: no-repeat;
verhindern. Möchte man allerdings das Gegenteil so kann kann man zum Beispiel nur horizontal wiederholen lassen mit
> background-image: url("gradient_bg.png"); background-repeat: repeat-x;

position festlegen Man kann die position festlegen mit
> background-image: url("beispiel.jpg");
background-repeat: no-repeat; background-position: right top;

die mitscrollen Möchte man das ein Hintergrund während des scrollen mitgeht so tut man dies mit
> background-image: url("beispiel.jpg"); background-attachment: scroll;

die an einer Stelle bleiben Möchte man das ein Hintergrund während des scrollen nicht mitgeht so tut man dies mit
> background-image: url("beispiel.jpg"); background-attachment: fixed;
